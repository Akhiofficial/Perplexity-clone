# Perplexity Clone

A full-stack web application designed as a clone of Perplexity AI. This project includes both a frontend and a backend, built to practice full-stack development and integrate AI search and metadata functionalities.

## Tech Stack

This project leverages modern, cutting-edge technologies to provide a seamless AI-powered search experience:

- **Frontend**: React, Vite
- **Backend**: Node.js, Express.js
- **AI & Search Tools**: **Tavily Search API** (for powerful, real-time web search discovery and extraction)

## Project Structure

- `/frontend`: Contains the client-side application (React / Vite).
- `/backend`: Contains the server-side application and API (Node.js / Express).

## Running the Application Locally

Both the frontend and backend need to be run concurrently for the full application to work.

### Frontend

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Backend

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure your `.env` variables (see the backend folder for required environment variables).
4. Start the server:
   ```bash
   npm run dev
   ```
